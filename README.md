# Harshit Farkare — Portfolio 🚀

A modern, high-end 3D portfolio website for an AI developer and automation engineer. Built using React, Three.js, Framer Motion, and deployed on Vercel.

## 🌟 Features
- **3D Interactive Elements**: Built with `@react-three/fiber` and `@react-three/drei`.
- **Smooth Animations**: Powered by `framer-motion`.
- **Modern UI**: Fully responsive, dark-themed style layout utilizing glassmorphism and smooth gradients.
- **Dynamic Content**: Showcasing projects, certifications, skills, and more.
- **Analytics Tracked**: Utilizing Vercel Analytics for tracking traffic.

## 🛠️ Technology Stack
- **Framework**: Vite + React
- **Styling**: Vanilla CSS (Modern Layout with CSS Variables)
- **3D Graphics**: Three.js
- **Animations**: Framer Motion
- **Icons**: React Icons
- **Deployment**: Vercel

## 🚀 Running Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/harshitfarkare63/harshitfarkare-portfolio.git
   ```
2. Navigate to the directory:
   ```bash
   cd harshitfarkare-portfolio
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
